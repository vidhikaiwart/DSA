# OOPS Example

This folder contains a simple C++ example for class and constructor concepts, including access modifiers, getters, and setters, and encapsulation.

- Class.cpp: demonstrates a basic class and object usage
- Constructor.cpp: contains constructor and destructor definitions
- Encapsulation.cpp: shows how data can be hidden using private members with getters and setters
- Access modifiers: shows how public and private members work
- Types of constructors: default, parameterized, copy, and destructor
